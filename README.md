본 저장소는 현대/기아/제네시스 전용 저장소입니다.

* 조향(lateral)과 크루즈 컨트롤(longitudinal)이 분리되어 따로 동작하게 되어 있으며 이는 콤마정책을 위반합니다.
* loggerd, uploader 프로세스를 활성화(기본적으로 비활성화)할 경우 기기가 콤마서버에서 차단될 수 있으니 주의하시기 바랍니다.
